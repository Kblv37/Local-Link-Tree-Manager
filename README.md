# Local Link Tree Manager

Локальное расширение для браузера, позволяющее хранить, структурировать и быстро находить ссылки в виде дерева папок — **без интернета и облаков**.

---

##  Возможности

-  Древовидная структура папок и ссылок  
-  Быстрый поиск в popup  
-  Глобальная горячая клавиша **Ctrl + Q / Cmd + Q**  
-  Drag & Drop для папок и ссылок  
-  Cut / Paste элементов дерева  
-  Ручное сохранение (без автосохранения)  
-  Привязка к JSON-файлу (чтение и запись)  
-  Синхронизация popup и страницы настроек  

---

##  Архитектура

- **Popup** — просмотр и поиск  
- **Options** — редактирование дерева  
- **Local Storage** — основной источник данных  
- **JSON-файл** — резервное хранилище / синхронизация  

---

##  Установка

1. Склонируйте репозиторий  
2. Откройте `chrome://extensions`  
3. Включите **Режим разработчика**  
4. Нажмите **Load unpacked** и выберите папку проекта  

---

##  Горячие клавиши

- **Ctrl + Q / Cmd + Q** — открыть поиск в popup  
- **Ctrl + S** — сохранить в настройках  
- **Esc** — отменить несохранённые изменения
