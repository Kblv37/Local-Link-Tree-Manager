📖 Документация
О расширении
  • Локальный менеджер ссылок в виде дерева
  • Папки могут вкладываться друг в друга
  • Данные хранятся только на вашем устройстве
🖱 Popup
  • Клик по папке — свернуть / развернуть
  • Клик по ссылке — открыть в новой вкладке
  • ⤴⤴ — открыть все ссылки папки
  • ⧉ — скопировать URL
  • ⊞ — свернуть / развернуть всё
⌨ Горячие клавиши
  • Alt+Q — открыть popup
  • ↑ ↓ — навигация по ссылкам
  • Enter — открыть выбранную ссылку
  • Esc — очистить поиск / снять выделение
  • Alt+S — сохранить дерево
📤 Экспорт / Импорт
  • «Экспорт» — скачать дерево как TXT
  • Флаг «уник. имя» — добавляет дату в имя файла
  • «Импорт» — загрузить TXT (новый и старый формат)
  • Настройки экспортируются отдельно
🛠 Редактор дерева
  • +Папка — добавить корневую папку
  • ✂ — вырезать, ⤓ — вставить в папку
  • ⬆⬇ — изменить порядок
  • 🧹 — удалить пустые папки
  • ↩ Отменить — восстановить из резервной копии
